درباره این پروژه

این وبسایت یک سیستم مدیریت آگهی کوچک است که با Django ساخته شده است. کاربران می‌توانند ثبت‌نام کنند، وارد شوند و آگهی‌های خود را ایجاد، ویرایش و حذف کنند. هر آگهی شامل عنوان، توضیحات، قیمت، شماره تماس و تصویر است.

هدف پروژه

هدف از ساخت این پروژه تمرین مهارت‌های Django و آماده‌سازی یک نمونه‌کار قابل ارائه به کارفرما بوده است.

تکنولوژی‌های استفاده شده

  • Python
  • Django
  • SQLite
  • Bootstrap (برای طراحی واکنش‌گرا)

ویژگی‌های پروژه

این پروژه شامل مدیریت کاربران، پیام موفقیت، آپلود تصویر و دسترسی محدود برای کاربران است و تمام قابلیت‌های پایه یک سایت ثبت آگهی واقعی را پوشش می‌دهد.
