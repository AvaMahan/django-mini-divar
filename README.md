# Django Mini Divar

A mini Divar-like classified ads platform built with Django.

## Features

- User registration and authentication
- Create, edit and delete ads
- Upload images for ads
- View ads list and ad details
- Access control based on login status

## Technologies

- Python
- Django
- HTML / CSS
- Bootstrap
- SQLite

## Installation

1. Clone the repository:
   git clone https://github.com/AvaMahan/django-mini-divar

2. Install dependencies:
   pip install -r requirements.txt

3. Apply migrations:
   python manage.py migrate

4. Run the development server:
   python manage.py runserver
